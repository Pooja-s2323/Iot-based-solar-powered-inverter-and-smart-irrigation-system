# IoT Based Solar Powered Inverter and Smart Irrigation System

## About Project
This project is developed for smart agriculture using IoT and solar energy.

## Features
- Automatic irrigation
- Soil moisture monitoring
- Rain detection
- Telegram alerts
- Solar powered system
- Animal detection using IR sensor

## Components Used
- ESP32
- DHT11 Sensor
- Soil Moisture Sensor
- Relay
- LCD
- Solar Panel
- Battery
